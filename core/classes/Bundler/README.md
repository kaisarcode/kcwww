# Bundler Utility

The `Bundler` utility provides methods to bundle and inline CSS and JavaScript files, resolving dependencies recursively.

## Methods

- `Bundler::css(string $file): string` - Bundles CSS files.
- `Bundler::js(string $file): string` - Bundles JS files.
